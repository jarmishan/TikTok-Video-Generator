# TikTok Video Generator

An automated content generation pipeline that creates short form quiz videos using music data.

## Overview

This project uses the Spotify API to identify artists within a genre and select tracks to generate short audio snippets. It then scrapes YouTube for relevant video content and combines everything into TikTok-style videos.

## Features

- Spotify API integration for artist and track selection  
- Automated YouTube scraping using Selenium  
- Audio processing and snippet generation  
- Video rendering using MoviePy  
- End-to-end automation from data collection to final video output  

## Impact

- Videos generated averaged ~3000 views per day
